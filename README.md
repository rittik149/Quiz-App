# 🧠 Online Quiz Application (Java)

This is a simple **Online Quiz Application** built using Java Swing. It allows users to take a multiple-choice quiz with a **timer** and **score tracking**.

---

## 🚀 Features

- 🧾 Multiple-choice questions (MCQs)
- ⏳ 15-second timer per question
- ✅ Automatic scoring
- 🔄 Auto-transition to the next question on timeout or answer
- 💡 Built with Java Swing for GUI


---

## 🛠️ How to Run

### Prerequisites:
- Java JDK installed (version 8 or above)
- IDE (like IntelliJ, Eclipse) or terminal/command prompt

### Steps:

1. **Clone or download the repository**

2. **Compile the code:**
   ```bash
   javac QuizApp.java
````

3. **Run the application:**

   ```bash
   java QuizApp
   ```

4. The quiz window will open. Answer the questions within 15 seconds each.
